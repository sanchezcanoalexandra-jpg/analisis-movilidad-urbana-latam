# analisis-movilidad-urbana-latam
# Contexto & Objetivo 
El propósito central de este análisis fue descubrir si la riqueza de una ciudad influye directamente en qué tan colapsadas están sus calles. Para lograrlo, cruzamos datos de movilidad urbana (como los minutos de retraso acumulados y el tiempo real que toma avanzar 10 kilómetros) con el indicador económico clave: el PIB per cápita (que representa el dinero que genera una ciudad por cada habitante). Entender esta relación es vital para los gobiernos y planeadores urbanos, ya que permite identificar si el crecimiento económico está ahogando la movilidad de una ciudad o si el problema se debe a un diseño deficiente de sus vías.
# Cobertura de Datos y Metodología 
Para tener una visión global, el estudio analizó registros del año 2024, cubriendo múltiples ciudades y países del mundo (incluyendo grandes capitales globales). El proceso comenzó con una limpieza para corregir nombres de columnas desorganizados y unificar los formatos. Después, agrupamos toda la información diaria para obtener un único promedio anual por cada ciudad. Finalmente, realizamos un cruce estricto (unión tipo INNER) para conservar únicamente las ciudades que contaban con información completa tanto de tráfico como de economía, validando todo el comportamiento mediante gráficos de distribución (cajas e histogramas) para asegurar que ningún dato extraño alterara las conclusiones generales.

# Hallazgos Iniciales 
El descubrimiento más importante es que no existe una regla de tres que dicte que a mayor riqueza, mayor tráfico. Ciudades con economías gigantescas y altos ingresos por habitante, como Tokio o Nueva York, logran mantener sus tiempos de viaje bajo un control aceptable. Por el contrario, ciudades con menores ingresos relativos, como Manila o Ciudad de México, muestran los peores colapsos viales y retrasos del estudio. Esto nos demuestra que el tráfico no es una consecuencia obligatoria de tener una economía fuerte, sino un síntoma de falta de alternativas de transporte masivo (como metros o trenes eficientes).

# Recomendaciones y Ciudad Prioritaria 
Basados en los resultados analizados en la región, la ciudad que se alinea perfectamente como la máxima prioridad de inversión es Ciudad de México (seguida muy de cerca en la tendencia regional por capitales de alta densidad como Lima o Bogotá si se evalúan bajo el mismo patrón de infraestructura rezagada). Estas ciudades muestran una correlación crítica: niveles de congestión vehicular alarmantes conviviendo con indicadores económicos que se ven frenados por las horas hombre desperdiciadas en el transporte.

La recomendación principal no es inyectar dinero a ciegas, sino priorizar la inversión en redes de transporte público subterráneo o exclusivo en estas ciudades específicas. Al sacar los autos de las calles principales mediante alternativas rápidas, no solo se reduce el estrés de los ciudadanos, sino que se destraba el potencial económico que hoy se pierde en las filas de tráfico. Como paso siguiente, se sugiere realizar un análisis más profundo enfocado exclusivamente en la densidad poblacional de estas zonas para diseñar las rutas de inversión exacta.
