# analisis-movilidad-urbana-latam
Contexto & Objetivo El propósito central de este análisis fue descubrir si la riqueza de una ciudad influye directamente en qué tan colapsadas están sus calles. Para lograrlo, cruzamos datos de movilidad urbana (como los minutos de retraso acumulados y el tiempo real que toma avanzar 10 kilómetros) con el indicador económico clave: el PIB per cápita (que representa el dinero que genera una ciudad por cada habitante). Entender esta relación es vital para los gobiernos y planeadores urbanos, ya que permite identificar si el crecimiento económico está ahogando la movilidad de una ciudad o si el problema se debe a un diseño deficiente de sus vías.
